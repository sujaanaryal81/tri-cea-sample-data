# Socioeconomics Data

County-level cost and economic variables feeding into CEA cost-benefit and systems dynamics modeling.

## Categories covered
- Utility costs (electricity, gas)
- Water costs
- Rental and land costs
- Fuel costs
- Local and state taxes
- Labor costs and working cash requirements
- Cost-benefit and budget tools for specific CEA structures

## Files
- lettuce_budget.csv - sample interactive cost and budget tool for a lettuce quonset-house operation

## Status
Baseline granularity is county-level. Finer granularity via ACS or Census data is a future direction.
