# Workforce Development Data

Labor market and training-pipeline data relevant to staffing CEA operations.

## Subfolders
- labor-market/ - regional workforce availability and wage data
- training-programs/ - workforce training program data
- community-college-partnerships/ - partnerships supporting CEA workforce pipelines
- workforce-costs/ - workforce cost estimates feeding the cost-benefit model

## Status
Targeted for the v2 data release (~May 2026), building on relationships with regional community colleges (e.g. Santa Fe Community College).
